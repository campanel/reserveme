# Reserveme

## Ambiente
composer create-project --prefer-dist laravel/laravel reserveme

chmod 777 -R storage/
chmod 777 -R bootstrap/cache/

mysql> create database reserveme;

Configurar .env com os dados do seu mysql

php artisan serve
